### SUBLEQ TOOLCHAIN

This is my big project to write a complete toolchain and implementation of the SUBLEQ instruction set architecture.

## It contains:
1. An emulator (python)
2. A assembler with extensive macro support (python)
3. A logism implementation
4. A system verilog implementation
5. A print circuit board implementation in kicad with photos (coming soon)

## What is SUBLEQ?
SUBLEQ is an ISA sometimes called OISC (one instruction set arch) that literally just does one instruction over and over, yet is still turing complete.
